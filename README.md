Reverse-engineering sandbox

## Commands

| Syntax                                     | Description |
|--------------------------------------------|-------------|
| q/quit                                     | Quit        |
| b/break <address>                          | Break at    |
| c/continue \[count]                        | Continue    |
| n/next \[count]                            | Step over   |
| s/step \[count]                            | Step into   |
| d/dump <offset> <count> --format \[format] | Dump        |
